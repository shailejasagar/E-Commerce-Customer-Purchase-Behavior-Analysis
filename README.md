# E-Commerce Customer Purchase Behavior Analysis
## Project Overview
This project dives into **E-Commerce Customer Purchase Behavior** using real-world transactional data.
### Objectives:
- Analyze customer purchasing patterns
- Perform RFM Segmentation
- Visualize monthly sales trends
- Identify top product categories
### Dataset
Custom E-Commerce Dataset with:
- Customer ID  
- Purchase Date  
- Product Category  
- Total Purchase Amount  
- Payment Method  
- Customer Age  
- Churn Status  
### Tech Stack
- Python  
- Pandas  
- Seaborn  
- Matplotlib  
- Jupyter Notebook  
### Results
- Monthly sales trends
- Top product categories
- Customer segments based on RFM scores
